# Bank Networking System (Cisco Packet Tracer)

## 📌 Description
This project simulates a real-world banking network infrastructure designed for expansion across multiple locations.  
The system is built using Cisco Packet Tracer and demonstrates efficient network design, routing, and security implementation.

---

## 🏢 Network Overview
- 4-floor building  
- Each floor contains 3 departments  
- Each department has multiple PCs and printers  
- All departments are connected to a central server  

---

## ✨ Features
- Centralized DHCP and DNS server  
- Inter-department communication  
- Secure network configuration  
- Wireless network setup  
- Scalable multi-floor architecture  

---

## 🛠️ Technologies & Tools
- Cisco Packet Tracer  
- Routers (Cisco 2911)  
- Switches (Layer 2: 2950, Layer 3: 3650)  
- VLANs, OSPF Routing Protocol  
- DHCP, DNS Configuration  

---

## 🧠 Concepts Implemented
- VLAN segmentation  
- Inter-VLAN routing  
- OSPF dynamic routing  
- Switch port security  
- SSH configuration for secure remote access  
- DHCP server configuration  
- Wireless networking  
- IP addressing and subnetting  

---

## ⚙️ Methodology
- Configured basic settings for all devices  
- Assigned VLANs and configured access/trunk ports  
- Implemented switch port security  
- Configured OSPF on routers and Layer 3 switches  
- Set up DHCP and DNS server  
- Enabled inter-VLAN routing using helper addresses  
- Configured wireless access points  
- Verified and tested network connectivity  

---

## ▶️ How to Run
1. Open Cisco Packet Tracer  
2. Load the `.pkt` file  
3. Run simulation mode  
4. Test connectivity between devices
